🏮 农历闹钟 · APK 打包指南

从 PWA 到 Android APK，全程免安装工具

1确认项目文件齐全

你的项目目录应该包含以下文件：

Claw/ ├── index.html ← 主页面（PWA 入口） ├── lunar-alarm.html ← 重定向到 index.html（向后兼容） ├── manifest.json ← PWA 配置 ├── sw.js ← Service Worker（离线缓存） └── icons/ ├── icon.svg ← SVG 矢量图标 ├── icon-maskable.svg ← 可遮罩 SVG 图标 ├── icon-72.png ← Android 小尺寸 ├── icon-96.png ← 快捷方式图标 ├── icon-128.png ├── icon-144.png ├── icon-152.png ← iOS 图标 ├── icon-192.png ← PWA 标准图标 ├── icon-384.png └── icon-512.png ← PWA 大图标 / 启动屏

✅ 以上文件已全部生成完毕。

2将项目部署到公网（必须是 HTTPS）

PWABuilder 需要通过公网 URL 访问你的 App，推荐免费方案：

  • Vercel（最简单）— 拖拽文件夹即可部署
  • GitHub Pages — 上传到仓库后启用 Pages
  • 腾讯云 CloudBase — 国内速度更快
📌 推荐：用 Vercel 拖拽部署
1. 打开 vercel.com → 注册/登录
2. 点击 Add New → Project → Browse（选文件夹方式）
3. 直接把整个 Claw 文件夹拖入上传区域（推荐！会自动保留子目录结构）
4. 几秒后获得类似 https://your-app.vercel.app 的地址
⚠️ 注意：「上传文件」按钮不支持子目录！
Vercel 文件管理里点「+」→「上传文件」只能传单层文件，无法自动创建 icons/ 子目录。
如果你已经点了这个按钮，正确做法是：
  ① 先点「+」→ 「新建文件夹」，输入 icons，创建好目录
  ② 进入 icons 文件夹后，再点「+」→「上传文件」，上传 SVG 和 PNG 图标
  ③ 返回根目录，继续上传根目录下的其他文件（index.html、sw.js 等）

💡 最省事的方式：整个文件夹直接拖拽到 Vercel 的拖拽区，一次搞定，不用手动建目录。
3用 PWABuilder 生成 APK

获取到 HTTPS 地址后：

  • 打开 pwabuilder.com
  • 在输入框粘贴你的网址，点击 Start
  • 等待分析完成（会给出 PWA 评分）
  • 点击 Package for Stores
  • 选择 Android
  • 点击 Generate Package
  • 下载生成的 .apk 文件
打开 PWABuilder →
💡 PWABuilder 生成的是基于 TWA（Trusted Web Activity） 的 APK， 本质上是一个 Chrome 容器运行你的网页，性能接近原生 App，完全免费。
4安装到手机
  • 将 .apk 文件传到 Android 手机（微信/QQ/USB 均可）
  • 在手机文件管理器找到该文件，点击安装
  • 如提示"未知来源"，在设置里允许即可
  • 安装后在桌面找到"农历闹钟"图标，打开即用！
⚠️ 注意：TWA 方案依赖手机上已安装的 Chrome（版本 ≥ 72）， 国内大多数 Android 手机都已满足。
5备选：直接用浏览器"添加到桌面"

如果不想折腾 APK，还有个零门槛方案：

  • 在 Android 手机的 Chrome 浏览器打开你的网址
  • 点击右上角 菜单（⋮）→ 添加到主屏幕
  • App 图标就出现在桌面了，体验和安装版一样
  • iOS 用 Safari 打开 → 分享 → 添加到主屏幕
✨ 这个方法连服务器都不需要，直接打开 HTML 文件即可在本地使用。
🎉

按以上步骤操作，大约 15 分钟即可完成 APK 打包！
